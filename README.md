# Statamic extended sections

![Statamic 3.0+](https://img.shields.io/badge/Statamic-3.0+-FF269E?style=for-the-badge&link=https://statamic.com)

> Statamic sections with fancy backgrounds

## Installation

Install with Composer:

    composer require appswithlove/statamic-extended-section

## Usage

Use the «Extended Section» field and choose the background you like in the blueprint editor.
